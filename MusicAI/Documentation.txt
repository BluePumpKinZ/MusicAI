-i		Creates a new network.

		-- Usage --
		-i [number of nodes] [number of connections] [number of inputs] [number of outputs]


-o		Opens an existing network.
		
		-- Usage --
		-o [path]


-s		Save the network to disk.
		
		-- Usage --
		-s [path]


-l		Let the network learn.

		-- Usage --
		-l [binary file (32 bit floating point)] [number of iterations]

		
-x		Let the network run.
		
		-- Usage --
		-x [binary file (32 bit floating point) path] [result path] [number of iterations]
